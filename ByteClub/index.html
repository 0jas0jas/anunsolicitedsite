About the club







Starting off as just a desire to bring together innovative minds and problem-solving thinkers, Byte_Club emerged as CMS Cambridge's very own code club. We found support and a recognisable platform with the help of HackClub (a global organisation for high school code clubs) which led us to finally being able to have an enthusiastic and diverse group of learners. Together, we are going to explore different areas of programming and Computer Science, learn SO MUCH cool stuff and mostly have TONS of fun together!
